# Sistem Kasir Botol
## Instalasi
### Masuk ke root project
``` bash
cd sistem-kasir-botol
```
### Copy environment
``` bash
cp includes/config.php.example includes/config.php
```
### Install Dependencies
``` bash
npm install
```
### Jalankan
Ada 2 kondisi
#### Development
Untuk development pakai:
```bash
npm run dev
```
#### Production
Untuk production pakai:
``` bash
npm run build
```
